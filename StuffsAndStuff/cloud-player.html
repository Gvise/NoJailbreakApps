Cloud Player is a screen recorder built for iOS 7. It records any part of your screen and is useful for iOS tutorials made for YouTube. It records very smoothly, with barely any lag or post-recording video glitches. It also records external sound with your device's microphone.

 

Note: Assistive Touch must be enabled and on to record OpenGLES games, like Minecraft PE.

 

This version should be pre-hacked, but if it doesn't work for you (only allowing 10 seconds of footage), download the file below and insert it into Cloud Player's Library>Preferences and allow it to replace the existing .plist. Do this with iFile if you're jailbroken or    i-FunBox if you're not.
